# JetpackJoyrideGame

## Instructions To Run Game:

* pip3 install -r requirements.txt
* python3 ./run.py

## Goal:

Make your way through the arena avoiding obstacles and collecting coins to increase score. At the end, defeat the boss to win the game!

## Move List:

* W - Move Up
* D - Move Right
* A - Move Left
* E - Shoot Bullets
* S - Activate Shield
* Q - Quit Game
